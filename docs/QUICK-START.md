# 🚀 快速上手指南(5分钟)

## 第一次使用？看这里

### 1️⃣ 了解这个Skill是什么(1分钟)
这是一个**电商产品上架自动化工作流**,可以把一个新品从零到可上架的所有准备工作(定位/竞品/主图/详情页/标题/客服话术等)压缩到一次会话内跑完。

**输入:** 产品名称+类目+平台+基础信息  
**输出:** 15项标准分析 + 4角色执行建议(运营/设计/客服/老板)  
**时间:** 3-8分钟

### 2️⃣ 复制调用模板(30秒)
打开 `templates/user-input-form.md`,复制这段:

```
【跑电商新品上架流程】

产品名称:
产品类目:
一句话描述:

目标平台:□ 淘宝 □ 拼多多 □ 亚马逊 □ 多平台

售价区间(元):
预估成本(元):

已知竞品(品牌 / 链接,选填):
1.
2.
3.

产品图(选填):□ 已附 □ 稍后补

我想特别强调的事(选填):

输出范围(选填,默认全跑):□ 全 15 项
```

### 3️⃣ 填写你的产品信息(2分钟)
**最少填3项(必填):**
- 产品名称:便携折叠咖啡杯
- 产品类目:餐饮具/杯具
- 目标平台:淘宝

**推荐再填3项(提升精度30%):**
- 售价区间:79-129元
- 预估成本:25元
- 已知竞品:Stanley / 酷革 / 哈尔斯

**最佳实践(精度最高):**
- 上传产品图(正面/背面/内部/上身/细节 5张)
- 给竞品链接而非品牌名
- 说明"重点想说的事"(如:主打办公室+通勤场景)

### 4️⃣ 发送给AI(等待3-8分钟)
粘贴填好的模板,AI会:

1. **验证输入** - 检查必填项,缺了会提示你补充
2. **声明数据真实性** - 告诉你哪些数据需要后期核验
3. **执行15项** - 实时显示进度(✅ 3/15 已完成)
4. **质量自检** - 自动检查11项关键指标
5. **输出4角色分工** - 运营/设计/客服/老板各自的待办清单

### 5️⃣ 查看输出(看懂15项)
**战略层(1-4项)** - 先定方向
- 1.产品定位 - 解决什么问题/哪个价格带
- 2.目标用户 - 谁买/决策路径
- 3.核心卖点 - 厂家语言→用户语言
- 4.用户痛点 - 真实场景里的烦

**市场层(5-6项)** - 找机会
- 5.竞品分析 - **必须分价格带**(至少3个)
- 6.差异化机会 - 3个可打的方向(带判断依据)

**视觉层(7-8项)** - 设计素材
- 7.主图策划 - 5张,每张带画面/文案/点击逻辑
- 8.详情页结构 - 按用户疑虑顺序排(不是卖点顺序)

**流量层(9-11项)** - 搜索优化
- 9.平台标题 - 淘宝/拼多多/抖音等分版本
- 10.关键词三层 - 核心词/场景词/卖点词
- 11.评论洞察 - 好评提炼+差评拆雷

**客服+协作层(12-14项)** - 落地执行
- 12.FAQ - 8-12个真实问题
- 13.客服话术 - 差评预防/售前答疑/售后安抚
- 14.设计Brief - 给设计师可执行的要求

**运营推进层(15项)** - 上架后优化
- 15.30天运营推进计划 - 7天测款/14天起量/30天稳定

**4角色分工** - 谁干什么
- 运营:决策点/优先级/删什么
- 设计:创意方向/雷区/风格参考
- 客服:重点答疑/情绪安抚/升级判断
- 老板:决策摘要/风险点/资源需求

## 💡 常见问题

### Q1: 必须提供产品图吗?
A: 不是必须,但**强烈推荐**。缺产品图会导致:
- 主图策划精度降低30%(只能描述,不能基于实际产品)
- 设计Brief缺少参考
- 差异化判断可能不准(不知道实际质感/细节)

### Q2: 竞品信息从哪找?
A: 3个来源:
1. 淘宝搜类目词,看前20个(按销量排序)
2. 用生意参谋/蝉妈妈/卖家精灵拉类目TOP100
3. 已知品牌+站内推荐

**给链接优于给品牌名** - Agent 会优先调用已安装的 companion skills 读取公开页面观察值,但页面展示价不等于真实成交价。

### Q3: AI生成的数据可信吗?
A: **分情况:**
- ✅ 战略判断/方法论/框架 - 可信度高(基于电商常识)
- ⚠️ 竞品价格/评论/搜索量 - 标注【待核验】,上架前必须用工具核实
- ❌ 生成的图片 - 仅作创意参考,不可直接上架

开始前会有"数据真实性声明",明确告诉你哪些需要核验。

### Q4: 15项太多,能只跑部分吗?
A: 可以,在调用模板里写:
```
输出范围:1,3,5,7(只跑定位/卖点/竞品/主图)
```
但**建议第一次全跑**,后续根据需要调整。

### Q5: 跑完后怎么用?
A: 4角色分工已经给出待办清单:
- **运营**拿着第1-6项去拍板(价格带/主打卖点/是否切细分),拿着第15项安排7天/14天/30天推进节奏
- **设计**拿着第7项和第14项去出图(主图5张+详情页+设计Brief)
- **客服**拿着第12/13项更新话术库
- **老板**拿着4角色最后一段去批资源(预算/人/时间)

可选:导出到飞书多维表格,团队协作更方便。

### Q6: 和原文章的区别?
A: 这是v1.0优化版:
- ✅ 增加了输入验证(不会空跑)
- ✅ 增加了数据真实性声明(知道哪些要核验)
- ✅ 增加了质量自检(11项指标,含文案人味化/生图参数确认/视频判断)
- ✅ 增加了工具自动调用(优先调用 companion search/scrape skills 发现竞品和读取公开页面观察值)
- ✅ 补全了平台规则(6大平台)
- ✅ 强化了差异化判断(3条标准+4列表格)
