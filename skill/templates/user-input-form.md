```
产品类目:箱包 / 通勤包

```

---

## ⚠️ 必填 3 项,缺一不可

1. **产品名称 + 一句话描述** — 没有这个,大管家不知道在分析什么
2. **产品类目** — 影响竞品选择和价格带判断
3. **目标平台** — 影响标题分版本和关键词分析

其他都是选填,但**给得越多,精度越高**。

## 💡 进阶调用

**想跳过某些项:**
```
输出范围:1,3,5,7(只跑定位/卖点/竞品/主图)
```

**想加新项:**
```
输出范围:1-15 + 16(包装设计)+ 17(短视频脚本)
```
> 新项必须符合"可执行 + 带为什么"两个约束,否则大管家会拒绝

**想同时跑 2 个产品对比:**
> 一条消息只跑 1 个。跑完第 1 个,再说"同样流程跑一下第 2 个:XX"
