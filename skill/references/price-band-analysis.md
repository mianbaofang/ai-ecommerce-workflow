# 竞品分价格带 · 方法论参考

> 这是 Codex 第二季里**最容易被忽略的硬约束**:"竞品必须分价格带"。不分 = 跑偏。

## 为什么必须分?

同一个品类,不同价格带是**完全不同的战场**:

| 价格带 | 打法 | 用户决策逻辑 |
|---|---|---|
| 49-79 元 | 低价大容量,主图"能装" | 比价,看销量,看差评 |
| 89-129 元 | 通勤电脑包,强调"防水 + 减负" | 看主图,看卖点,看评论 |
| 139-199 元 | 商务通勤,强调"版型 + 质感" | 看详情页,看品牌感 |
| 199+ 元 | 品牌感 + 面料 + 设计 | 信任品牌,看细节 |

**不分价格带写竞品,会得到一堆空话:**
> "竞品都强调大容量和防水" — 这是低价带的现象,中高端根本不是这么打

**分价格带之后,差异化机会自己就出来了:**
> 109-159 元带:竞品同质化严重,主图都长一样 → 机会在"具体场景化"

## 实战 3 步法

### 第 1 步:找竞品

**3 个来源:**
1. **淘宝/拼多多搜品类词** — 排序按销量,看前 20 个
2. **蝉妈妈 / 卖家精灵 / 生意参谋** — 拉类目 TOP 100
3. **已知品牌 + 站内推荐** — 顺藤摸瓜

**每个价格带至少找 3-5 个代表。**

### 第 2 步:分带标签

按**实际成交价**分,不是按标价分(很多标价虚高)。

| 价格带 | 数量 | 代表竞品 | 共性打法 | 普遍问题 |
|---|---|---|---|---|
| 49-79 元 | 列出 | A / B / C | 低价 + 容量 | 质感 / 异味 |
| 89-129 元 | 列出 | D / E / F | 通勤 + 防水 | 同质化 / 主图雷同 |
| 139-199 元 | 列出 | G / H | 商务 + 版型 | 场景弱 |
| 199+ 元 | 列出 | I / J | 品牌 + 面料 | 转化依赖品牌 |

### 第 3 步:找空白带

**问 3 个问题:**
1. **哪个带最挤?** — 89-129 是大部分品类的红海
2. **哪个带用户最多但供给少?** — 通常是 109-159(中产消费力 + 品质敏感)
3. **我有能力做哪个带?** — 能力 = 品牌 / 供应链 / 资金

**空白带特征:**
- 竞品主图都长一样
- 好评集中在"够用"而不是"超预期"
- 差评集中在一个共性痛(品质 / 细节 / 售后)

## 案例:通勤包价格带分析

(参考 `examples/commute-backpack.md` 第 5 项)

| 价格带 | 打法 | 问题 |
|---|---|---|
| 49-79 | 低价大容量 | 质感弱,差评集中做工和异味 |
| 89-129 | 通勤电脑包 | 卖点同质化,主图差不多 |
| 139-199 | 商务通勤 | 场景强但功能讲得不够 |
| 199+ | 品牌感 | 转化依赖品牌信任 |

**机会带判断:** 109-159 元
- 比 89-129 多了质感升级空间
- 比 199+ 不依赖品牌信任
- 用户愿意为"通勤 + 短途两用"场景多付 30-50 元

## ⚠️ 常见错误

1. **价格带只分 2 段**(低 / 高) — 太粗,机会藏在中间
2. **跨类目比较**(用"包"对标"鞋") — 决策逻辑完全不同
3. **只看自己所在的带** — 看不到上下游挤压
4. **不验证成交价** — 标 199 的包,实际成交 89,属于中间带
5. **分完价格带就结束** — 一定要回到"我打哪个带、为什么"
